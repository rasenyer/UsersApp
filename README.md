# UsersApp
Esta es una simple aplicación Android, que lee datos de usuarios de una base de datos local, previamente autocompletada con Room, usando la arquitectura MVVM.

<h3>Librerías usadas:</h3>
• Coil
<br>
• Room
<br>
• Viewmodel
<br>
• Livedata
<br>
• Coroutines
<br>
• Navigation Component
